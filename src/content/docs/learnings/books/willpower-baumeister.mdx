## Power BI Fleet GHG KPI Dashboard — Complete Build Guide

### Dashboard Purpose

- Monitor fleet GHG emissions across multiple vessel fleets and years
- Track fuel consumption by type and mix shift over time
- Compare actual versus forecast performance
- Track progress toward 2026 and 2030 decarbonisation targets
- Enable vessel-level and ship-class-level drill-down analysis

### Data Sources

Raw Excel files stored in SharePoint under the following folder structure:

```
CMG_GHG_KPIs/
+-- Analysis/
|   +-- CMG_GHG_Master.pbix        (Power BI output file)
+-- Incoming/
    +-- AIDA/
    |   +-- 2019/
    |   |   +-- AIDA_FY19_ACT.xlsx
    |   +-- 2023/
    |   |   +-- AIDA_FY23_ACT.xlsx
    |   +-- 2026/
    |       +-- F01_FY2026.xlsx
    |       +-- F02_FY2026.xlsx
    |       +-- ...
    +-- Costa/
        +-- 2019/
        |   +-- COSTAFY19ACT.xlsx
        +-- 2026/
            +-- COSTAFY26ACT.xlsx
            +-- F01_FY2026.xlsx
```

### File Naming Rules

| Fleet | ACT file pattern | Example |
|---|---|---|
| AIDA | `AIDA_FY##_ACT.xlsx` | AIDA_FY23_ACT.xlsx |
| Costa | `COSTAFY##ACT.xlsx` | COSTAFY23ACT.xlsx |
| Any fleet forecast | `F##_FY####.xlsx` | F05_FY2026.xlsx |

### File Inclusion Rules

| Year folder | Data type | Include? |
|---|---|---|
| Current year | Forecast files | YES — all forecast versions |
| Current year | ACT files | NO |
| Older years | ACT files | YES |
| Older years | Forecast files | NO |

### Fiscal Year Calendar

Fiscal year runs December (year-1) to November (year).

| Folder year | FY months |
|---|---|
| 2019 | Dec 2018 — Nov 2019 |
| 2026 | Dec 2025 — Nov 2026 |

## Raw Excel File Structure

Each workbook has exactly one sheet. Structure is identical across all files and fleets.

| Row / Column | Content |
|---|---|
| Row 4, H–S | Month names: DEC, JAN, FEB, MAR, APR, MAY, JUN, JUL, AUG, SEP, OCT, NOV |
| Row 6, Column C | `# DAYS` — days row, skip |
| Row 8, Column D | Parameter block header name (e.g. ALBDs FULL MONTH) |
| Row 8, Column E | Label: `SHIP CLASS` |
| Row 8, Column F | Label: `ALB` |
| Row 8, H–S | Fleet totals — exclude |
| Row 9+, Column C | Vessel code (alphanumeric like LO346) |
| Row 9+, Column D | Vessel name |
| Row 9+, Column E | Ship class value |
| Row 9+, Column F | ALB value (Available Lower Berths) |
| Row 9+, H–S | Monthly parameter values |

**Important notes:**
- Column A contains Show/Hide/Check labels — ignore
- Columns T onwards contain yearly/quarterly totals — ignore
- Zero-value vessel rows may be hidden in Excel but are present — include only if Column D has a valid vessel name
- ALBDs FULL MONTH is always the first parameter block — Ship Class and ALB extracted from this block only
- Parameter block header row identified by Column E = `SHIP CLASS`
- Fleet total row identified by Column C being purely numeric (e.g. 993020)

### Query 1 — File_List

Handles SharePoint connection, file filtering, and metadata extraction. Binary content buffered **after** inclusion filter so only required files are downloaded.

**Enable Load:** Unchecked (connection only)

```
let
    Source = SharePoint.Files(
        "https://yourcompany.sharepoint.com/sites/YourSiteName",
        [ApiVersion = 15]
    ),

    FilteredToIncoming = Table.SelectRows(
        Source,
        each Text.Contains([Folder Path], "CMG_GHG_KPIs/Incoming")
    ),

    FilteredToExcel = Table.SelectRows(
        FilteredToIncoming,
        each [Extension] = ".xlsx" or [Extension] = ".xls"
    ),

    RenameMetaCols = Table.RenameColumns(
        FilteredToExcel,
        {
            {"Folder Path", "FolderPath"},
            {"Date modified", "DateModified"},
            {"Date created", "DateCreated"}
        }
    ),

    FixDates = Table.TransformColumns(
        RenameMetaCols,
        {
            {"DateModified",
                each if _ = null then null
                     else DateTime.From(
                         DateTimeZone.RemoveZone(
                             DateTimeZone.From(_))),
                type datetime},
            {"DateCreated",
                each if _ = null then null
                     else DateTime.From(
                         DateTimeZone.RemoveZone(
                             DateTimeZone.From(_))),
                type datetime}
        }
    ),

    AddPathParts = Table.AddColumn(
        FixDates, "Path_Parts",
        each Text.Split([FolderPath], "/")
    ),

    AddIncomingIndex = Table.AddColumn(
        AddPathParts, "Incoming_Index",
        each List.PositionOf([Path_Parts], "Incoming")
    ),

    AddFleet = Table.AddColumn(
        AddIncomingIndex, "Fleet",
        each if [Incoming_Index] >= 0
             then [Path_Parts]{[Incoming_Index] + 1}
             else "Unknown"
    ),

    AddYearFolder = Table.AddColumn(
        AddFleet, "Year_Folder",
        each
            let YearText = [Path_Parts]{[Incoming_Index] + 2}
            in  if Value.Is(Value.FromText(YearText), type number)
                then Number.FromText(YearText)
                else null
    ),

    AddDataType = Table.AddColumn(
        AddYearFolder, "Data_Type",
        each if Text.Contains(Text.Upper([Name]), "ACT") then "ACT"
             else if Text.Start([Name], 1) = "F"
                  and Text.Contains([Name], "_FY") then "Forecast"
             else "Unknown"
    ),

    AddForecastNumber = Table.AddColumn(
        AddDataType, "Forecast_Number",
        each if [Data_Type] = "Forecast"
             then Text.Middle([Name], 1, 2)
             else null
    ),

    AddFYYear = Table.AddColumn(
        AddForecastNumber, "FY_Year",
        each if [Data_Type] = "Forecast"
             then
                let
                    FYPos    = Text.PositionOf([Name], "_FY"),
                    YearText = Text.Middle([Name], FYPos + 3, 4)
                in
                    if Value.Is(Value.FromText(YearText), type number)
                    then Number.FromText(YearText)
                    else null
             else [Year_Folder]
    ),

    CurrentYear = Date.Year(DateTime.LocalNow()),

    AddIncludeFlag = Table.AddColumn(
        AddFYYear, "Include_File",
        each
            if [Year_Folder] = CurrentYear
               and [Data_Type] = "Forecast" then true
            else if [Year_Folder] < CurrentYear
               and [Data_Type] = "ACT" then true
            else false
    ),

    FilterIncluded = Table.SelectRows(
        AddIncludeFlag,
        each [Include_File] = true
    ),

    RemoveHelpers = Table.RemoveColumns(
        FilterIncluded,
        {
            "Path_Parts", "Incoming_Index",
            "Include_File", "Extension",
            "Attributes", "Date accessed",
            "FolderPath"
        }
    ),

    BufferContent = Table.TransformColumns(
        RemoveHelpers,
        {"Content", each Binary.Buffer(_)}
    ),

    FinalBuffer = Table.Buffer(BufferContent)

in
    FinalBuffer
```

### Query 2 — cmg_ghg_master_query

Main transformation query. All file processing happens inline.

**Key corrections applied:**
- `ALB` column explicitly converted to `Int64.Type` after vessel row filtering
- `Nautical Miles [NM]` renamed to `Nautical Miles` to avoid DAX bracket conflicts
- Null values replaced with 0 for all parameter columns after pivot
- `ALB` column type fixed at outer level

**Enable Load:** Checked (loads into Power BI model)

```
let
    FileList = Table.Buffer(File_List),
    RefreshTime = DateTime.LocalNow(),

    AddProcessed = Table.AddColumn(
        FileList,
        "ProcessedData",
        each
            let
                FYYear_v   = [FY_Year],
                Fleet_v    = [Fleet],
                Year_v     = [Year_Folder],
                DataType_v = [Data_Type],
                FcstNum_v  = [Forecast_Number],
                FileName_v = [Name],
                DateMod_v  = [DateModified],
                DateCre_v  = [DateCreated],

                Wb          = Excel.Workbook([Content], null, true),
                SheetName_v = Wb{0}[Name],
                SheetData   = Wb{0}[Data],

                CleanData = Table.ReplaceErrorValues(
                    SheetData,
                    List.Transform(
                        Table.ColumnNames(SheetData),
                        each {_, null}
                    )
                ),

                FirstSheet = Table.Buffer(CleanData),

                MonthRow = Table.SelectRows(
                    FirstSheet,
                    each [Column8] = "DEC"
                ),

                M1  = Record.Field(MonthRow{0}, "Column8"),
                M2  = Record.Field(MonthRow{0}, "Column9"),
                M3  = Record.Field(MonthRow{0}, "Column10"),
                M4  = Record.Field(MonthRow{0}, "Column11"),
                M5  = Record.Field(MonthRow{0}, "Column12"),
                M6  = Record.Field(MonthRow{0}, "Column13"),
                M7  = Record.Field(MonthRow{0}, "Column14"),
                M8  = Record.Field(MonthRow{0}, "Column15"),
                M9  = Record.Field(MonthRow{0}, "Column16"),
                M10 = Record.Field(MonthRow{0}, "Column17"),
                M11 = Record.Field(MonthRow{0}, "Column18"),
                M12 = Record.Field(MonthRow{0}, "Column19"),

                KeepCols = Table.SelectColumns(FirstSheet, {
                    "Column1", "Column3", "Column4",
                    "Column5", "Column6",
                    "Column8",  "Column9",  "Column10",
                    "Column11", "Column12", "Column13",
                    "Column14", "Column15", "Column16",
                    "Column17", "Column18", "Column19"
                }),

                R1 = Table.SelectRows(KeepCols,
                    each [Column8] <> "DEC"),
                R2 = Table.SelectRows(R1,
                    each [Column3] <> "# DAYS"),
                R3 = Table.SelectRows(R2,
                    each [Column1] <> "Check"),
                R4 = Table.SelectRows(R3,
                    each not (
                        [Column3] = null and [Column4] = null
                    )),

                AddPar = Table.AddColumn(R4, "Parameter",
                    each if [Column5] = "SHIP CLASS"
                         then Text.From([Column4])
                         else null),
                FillPar = Table.FillDown(AddPar, {"Parameter"}),

                AddSC = Table.AddColumn(FillPar, "Ship_Class",
                    each if [Parameter] = "ALBDs FULL MONTH"
                            and [Column5] <> "SHIP CLASS"
                            and [Column5] <> null
                            and Text.From([Column5]) <> "0"
                         then Text.From([Column5])
                         else null),

                AddALB_col = Table.AddColumn(AddSC, "ALB",
                    each if [Parameter] = "ALBDs FULL MONTH"
                            and [Column6] <> null
                            and [Column6] <> 0
                         then [Column6]
                         else null),

                FillSALB = Table.FillDown(
                    AddALB_col, {"Ship_Class", "ALB"}),

                VesselRows = Table.SelectRows(FillSALB,
                    each
                        [Column4] <> null
                        and [Column4] <> 0
                        and Text.From([Column4]) <> "0"
                        and Text.From([Column4]) <> "0.00"
                        and [Column5] <> "SHIP CLASS"
                        and not (
                            Text.Length(
                                Text.Remove(
                                    Text.From([Column3]),
                                    {"0".."9"}
                                )
                            ) = 0
                        )
                ),

                ConvertALB = Table.TransformColumns(
                    VesselRows,
                    {
                        "ALB",
                        each if _ = null then null
                             else try Int64.From(_) otherwise null,
                        Int64.Type
                    }
                ),

                Renamed = Table.RenameColumns(ConvertALB, {
                    {"Column3", "Vessel_Code"},
                    {"Column4", "Vessel_Name"},
                    {"Column5", "SC_Raw"},
                    {"Column6", "ALB_Raw"},
                    {"Column8",  M1},  {"Column9",  M2},
                    {"Column10", M3},  {"Column11", M4},
                    {"Column12", M5},  {"Column13", M6},
                    {"Column14", M7},  {"Column15", M8},
                    {"Column16", M9},  {"Column17", M10},
                    {"Column18", M11}, {"Column19", M12}
                }),

                Dropped = Table.RemoveColumns(Renamed,
                    {"Column1", "Vessel_Code",
                     "SC_Raw", "ALB_Raw"}),

                Unpivoted = Table.UnpivotOtherColumns(
                    Dropped,
                    {"Vessel_Name", "Ship_Class",
                     "ALB", "Parameter"},
                    "Month", "Value"),

                Buf       = Table.Buffer(Unpivoted),
                ParamList = List.Buffer(
                    List.Distinct(Buf[Parameter])),

                Pivoted = Table.Pivot(
                    Buf, ParamList,
                    "Parameter", "Value", List.Sum),

                ReplaceNulls = Table.TransformColumns(
                    Pivoted,
                    List.Transform(
                        ParamList,
                        each {
                            _,
                            each if _ = null then 0 else _,
                            type number
                        }
                    )
                ),

                RenameNM =
                    let
                        Cols = Table.ColumnNames(ReplaceNulls)
                    in
                        if List.Contains(Cols, "Nautical Miles [NM]")
                        then Table.RenameColumns(ReplaceNulls,
                            {{"Nautical Miles [NM]", "Nautical Miles"}})
                        else if List.Contains(Cols, "NAUTICAL MILES [NM]")
                        then Table.RenameColumns(ReplaceNulls,
                            {{"NAUTICAL MILES [NM]", "Nautical Miles"}})
                        else ReplaceNulls,

                WithYear = Table.AddColumn(RenameNM,
                    "Calendar_Year",
                    each if [Month] = "DEC"
                         then FYYear_v - 1
                         else FYYear_v),

                WithMNum = Table.AddColumn(WithYear,
                    "Month_Number",
                    each if [Month] = "JAN" then 1
                         else if [Month] = "FEB" then 2
                         else if [Month] = "MAR" then 3
                         else if [Month] = "APR" then 4
                         else if [Month] = "MAY" then 5
                         else if [Month] = "JUN" then 6
                         else if [Month] = "JUL" then 7
                         else if [Month] = "AUG" then 8
                         else if [Month] = "SEP" then 9
                         else if [Month] = "OCT" then 10
                         else if [Month] = "NOV" then 11
                         else if [Month] = "DEC" then 12
                         else null),

                A1  = Table.AddColumn(WithMNum,
                          "Source_File",      each FileName_v),
                A2  = Table.AddColumn(A1,
                          "Sheet_Name",       each SheetName_v),
                A3  = Table.AddColumn(A2,
                          "Date_Modified",    each DateMod_v),
                A4  = Table.AddColumn(A3,
                          "Date_Created",     each DateCre_v),
                A5  = Table.AddColumn(A4,
                          "Refresh_DateTime", each RefreshTime),
                A6  = Table.AddColumn(A5,
                          "Fleet",            each Fleet_v),
                A7  = Table.AddColumn(A6,
                          "Year_Folder",      each Year_v),
                A8  = Table.AddColumn(A7,
                          "Data_Type",        each DataType_v),
                A9  = Table.AddColumn(A8,
                          "Forecast_Number",  each FcstNum_v),
                A10 = Table.AddColumn(A9,
                          "FY_Year",          each FYYear_v),

                FixedCols = {
                    "Source_File", "Sheet_Name",
                    "Date_Modified", "Date_Created",
                    "Refresh_DateTime", "Fleet",
                    "Year_Folder", "Data_Type",
                    "Forecast_Number", "FY_Year",
                    "Vessel_Name", "Ship_Class", "ALB",
                    "Calendar_Year", "Month", "Month_Number"
                },

                Reordered = Table.ReorderColumns(A10,
                    FixedCols &
                    List.Difference(
                        Table.ColumnNames(A10), FixedCols))

            in
                Reordered
    ),

    CombinedTable = Table.Combine(
        AddProcessed[ProcessedData]),

    FixColumnTypes = Table.TransformColumnTypes(
        CombinedTable,
        {{"ALB", Int64.Type}}
    ),

    SortedTable = Table.Sort(
        FixColumnTypes,
        {
            {"Fleet",        Order.Ascending},
            {"FY_Year",      Order.Ascending},
            {"Vessel_Name",  Order.Ascending},
            {"Month_Number", Order.Ascending}
        }
    )

in
    SortedTable
```

```
Year_Version =
VAR YearText = TEXT([FY_Year], "0")

    [Data_Type] = "ACT",

### Known Type Issues

Two columns come through as text type from Power Query. Use `VALUE()` wrapper in DAX wherever used in comparisons or arithmetic.

| Column | Issue | DAX Fix |
|---|---|---|
| `ALB` | Text type | `VALUE(MAX([ALB]))` |
| `FY_Year` | Text type | `VALUE(MAX([FY_Year]))` or `VALUE([FY_Year])` in FILTER |

### Section F — Year Over Year Measures

```
kgCO2e ALBD Previous Year =
VAR CY = VALUE(MAX('cmg_ghg_master_query'[FY_Year]))
RETURN
CALCULATE(
    [kgCO2e per ALBD],
    FILTER(
        ALL('cmg_ghg_master_query'),
        VALUE('cmg_ghg_master_query'[FY_Year]) = CY - 1
    )
)
```

```
kgCO2e ALBD YoY Change =
[kgCO2e per ALBD] - [kgCO2e ALBD Previous Year]
```

```

## Maintenance Notes

**Adding a new fleet:** `File_List` dynamically detects fleet names from folder structure. Create new fleet folder inside `CMG_GHG_KPIs/Incoming/` on SharePoint — picked up automatically on next refresh. No code changes needed.

**Adding new target values:** Home → Transform Data → select `Targets` table → add rows directly.

**Updating emission factors:** Home → Transform Data → select `Emission_Factors` table → update `EF_tCO2e_per_MT` values. All CO2e measures recalculate automatically.

**Adding new parameters from source files:** Power Query pivot step automatically creates new columns for new parameters. Add corresponding measure in `_Measures` following existing fuel measure pattern. Update `Emission_Factors` table with new fuel type and factor.
